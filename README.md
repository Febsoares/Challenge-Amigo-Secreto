## Descrição

O **Amigo Secreto** é uma aplicação web que facilita a organização de amigos secretos entre grupos de pessoas. Com uma interface intuitiva, os usuários podem criar grupos, adicionar participantes e realizar o sorteio de forma automática, garantindo imparcialidade e diversão.

## Funcionalidades

- **Adição de Participantes**: Os usuários devem adicionar participantes manualmente.
- **Sorteio Automático**: Realiza o sorteio de forma automática, ao clicar no botão "sortear amigo".
